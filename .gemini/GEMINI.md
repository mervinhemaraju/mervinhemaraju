# My Global Developer Preferences

## Languages & Stack

- Primary languages: Dart, Kotlin, Python, YAML, JSON, Terraform, Go, Rust
- Mobile: Flutter (prefer latest stable SDK conventions)
- Backend: Python, Docker, Kubernetes
- Always use null safety in Dart
- Prefer `const` constructors in Flutter wherever possible

## Working Style

- Ask before making architectural changes
- Show me a plan for anything touching > 3 files
- Prefer small, focused changes — no sweeping rewrites unless asked
- When in doubt, show me options rather than choosing silently
- Explain *why* when you make a non-obvious decision
