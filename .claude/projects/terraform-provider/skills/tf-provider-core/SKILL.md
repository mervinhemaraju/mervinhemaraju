## HashiCorp Provider Design Principles (Official)

These are non-negotiable for any provider targeting the Terraform Registry.

- **One provider = one API or problem domain.** Do not mix multiple APIs into one provider.
- **One resource = one API object.** Resources are declarative representations of a single component with CRUD methods. Never abstract multiple API objects into one resource — use Terraform modules for that.
- **Schema must closely mirror the API.** Use the same field names and structure as the upstream API, unless it creates a genuinely bad UX. Do not invent Terraform-specific abstractions.
- **Every resource must support `terraform import`.** No exceptions.
- **Dates and times must be RFC 3339** strings (e.g. `2024-01-15T10:30:00Z`).
- **Boolean attributes: `true` means DO, `false` means DON'T.** Never use negative flags (e.g. prefer `enable_logging` over `disable_logging`).
- **Functions must be pure and offline.** No network calls, no time-based logic. Use data sources for those.
- **Ephemeral resources for sensitive API objects.** API tokens, secrets, and credentials that must not be stored in state should be modeled as ephemeral resources, not regular resources or data sources.

## Project Structure

```
terraform-provider-<name>/
├── internal/
│   ├── provider/
│   │   ├── provider.go
│   │   └── provider_test.go
│   ├── resources/
│   │   ├── <resource>_resource.go
│   │   └── <resource>_resource_test.go
│   ├── datasources/
│   │   ├── <resource>_data_source.go
│   │   └── <resource>_data_source_test.go
│   ├── ephemeral/
│   │   └── <resource>_ephemeral.go   # for sensitive API objects (tokens etc.)
│   └── client/
│       └── client.go                  # all HTTP logic lives here
├── examples/
│   └── resources/
│       └── <resource>/
│           └── resource.tf
├── docs/                              # generated by tfplugindocs — do not edit manually
├── templates/                         # tfplugindocs templates (optional)
├── main.go
├── GNUmakefile
├── CHANGELOG.md
└── .goreleaser.yml
```

Rules:

- One file per resource, data source, or ephemeral resource
- `internal/` package — nothing here is exported to consumers
- `client/` package contains all HTTP/API logic — never inline HTTP calls in resource files
- `docs/` is always generated, never hand-edited
- `examples/` must have a working `.tf` example for every resource

## main.go

```go
package main

import (
	"context"
	"flag"
	"log"

	"github.com/hashicorp/terraform-plugin-framework/providerserver"
	"github.com/<org>/terraform-provider-<name>/internal/provider"
)

// version is set by GoReleaser at build time
var version string = "dev"

func main() {
	var debug bool
	flag.BoolVar(&debug, "debug", false, "run provider with debugger support")
	flag.Parse()

	opts := providerserver.ServeOpts{
		Address: "registry.terraform.io/<org>/<name>",
		Debug:   debug,
	}

	err := providerserver.Serve(context.Background(), provider.New(version), opts)
	if err != nil {
		log.Fatal(err.Error())
	}
}
```

## provider.go

```go
package provider

import (
	"context"
	"os"

	"github.com/hashicorp/terraform-plugin-framework/datasource"
	"github.com/hashicorp/terraform-plugin-framework/ephemeral"
	"github.com/hashicorp/terraform-plugin-framework/provider"
	"github.com/hashicorp/terraform-plugin-framework/provider/schema"
	"github.com/hashicorp/terraform-plugin-framework/resource"
	"github.com/hashicorp/terraform-plugin-framework/types"
	"github.com/<org>/terraform-provider-<name>/internal/client"
)

var _ provider.Provider = &MyProvider{}

type MyProvider struct {
	version string
}

type providerModel struct {
	Host   types.String `tfsdk:"host"`
	APIKey types.String `tfsdk:"api_key"`
}

func New(version string) func() provider.Provider {
	return func() provider.Provider {
		return &MyProvider{version: version}
	}
}

func (p *MyProvider) Metadata(_ context.Context, _ provider.MetadataRequest, resp *provider.MetadataResponse) {
	resp.TypeName = "<name>"
	resp.Version = p.version
}

func (p *MyProvider) Schema(_ context.Context, _ provider.SchemaRequest, resp *provider.SchemaResponse) {
	resp.Schema = schema.Schema{
		Description: "Manages resources via the <Name> API.",
		Attributes: map[string]schema.Attribute{
			"host": schema.StringAttribute{
				Optional:    true,
				Description: "Base URL of the API. Can also be set via the <NAME>_HOST environment variable.",
			},
			"api_key": schema.StringAttribute{
				Optional:    true,
				Sensitive:   true,
				Description: "API key for authentication. Can also be set via the <NAME>_API_KEY environment variable.",
			},
		},
	}
}

func (p *MyProvider) Configure(ctx context.Context, req provider.ConfigureRequest, resp *provider.ConfigureResponse) {
	var config providerModel
	resp.Diagnostics.Append(req.Config.Get(ctx, &config)...)
	if resp.Diagnostics.HasError() {
		return
	}

	// Prefer env vars over config — standard Terraform provider UX pattern
	host := os.Getenv("<NAME>_HOST")
	if !config.Host.IsNull() {
		host = config.Host.ValueString()
	}
	apiKey := os.Getenv("<NAME>_API_KEY")
	if !config.APIKey.IsNull() {
		apiKey = config.APIKey.ValueString()
	}

	if host == "" {
		resp.Diagnostics.AddError(
			"Missing API Host",
			"Set the host provider attribute or <NAME>_HOST environment variable.",
		)
	}
	if apiKey == "" {
		resp.Diagnostics.AddError(
			"Missing API Key",
			"Set the api_key provider attribute or <NAME>_API_KEY environment variable.",
		)
	}
	if resp.Diagnostics.HasError() {
		return
	}

	c := client.New(host, apiKey)
	resp.ResourceData = c
	resp.DataSourceData = c
}

func (p *MyProvider) Resources(_ context.Context) []func() resource.Resource {
	return []func() resource.Resource{
		// Register resources here e.g. resources.NewThingResource
	}
}

func (p *MyProvider) DataSources(_ context.Context) []func() datasource.DataSource {
	return []func() datasource.DataSource{
		// Register data sources here
	}
}

func (p *MyProvider) EphemeralResources(_ context.Context) []func() ephemeral.EphemeralResource {
	return []func() ephemeral.EphemeralResource{
		// Register ephemeral resources here (tokens, secrets, credentials)
	}
}
```

## GNUmakefile

```makefile
HOSTNAME=registry.terraform.io
NAMESPACE=<org>
NAME=<name>
BINARY=terraform-provider-${NAME}
VERSION=0.1.0

default: fmt lint build

build:
	go build -o ${BINARY}

install: build
	mkdir -p ~/.terraform.d/plugins/${HOSTNAME}/${NAMESPACE}/${NAME}/${VERSION}/darwin_amd64
	mv ${BINARY} ~/.terraform.d/plugins/${HOSTNAME}/${NAMESPACE}/${NAME}/${VERSION}/darwin_amd64

fmt:
	gofmt -s -w .
	goimports -w .

vet:
	go vet ./...

lint:
	golangci-lint run ./...

generate:
	go generate ./...
	tfplugindocs generate

testacc:
	TF_ACC=1 go test ./... -v -timeout 120m

testunit:
	go test ./... -v -run "^TestUnit"

.PHONY: build install fmt vet lint generate testacc testunit
```
