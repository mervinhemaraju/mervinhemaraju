# Git — Hands Off

**Never run any git commit, git push, git merge, or git rebase commands.**
**Never stage files with git add unless I explicitly ask you to.**

I manage all git operations myself. Your job ends at writing and editing files.

You MAY use these read-only git commands to gather context:

- `git status` — check what has changed
- `git diff` — inspect changes
- `git log --oneline` — understand history
- `git branch` — check current branch

If you think something should be committed, tell me what and why — then stop.
